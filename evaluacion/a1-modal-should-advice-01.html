Modal Verb 'Should' — Advice & Suggestions

A1 — grammar, vocabulary, listening, reading, writing

Common mistake: “You should to try the fish.” — Correct: “You should try the fish.”

We use 'should' to give advice and suggestions; should + the base verb (no 'to').

should — advice
FormExample
AffirmativeYou should rest
NegativeYou shouldn't smoke
QuestionShould I go?
Short answerYes, you should.
  • try the local dish — probar el plato típico
  • book a table — reservar mesa
  • arrive early — llegar temprano
  • order dessert — pedir postre
  • ask the waiter — preguntar al mesero
  • leave a tip — dejar propina
  • You should try the local food. We should book a table for tonight.
  • You should arrive early because it gets busy.
  • You shouldn't arrive late, and you shouldn't forget the tip.
  • We shouldn't order too much food.
  • Should I order dessert? What should we try first?
  • Should we ask the waiter for a recommendation?
  • Should I book a table? Yes, you should.
  • Should we skip dessert? No, you shouldn't.

Practice

  1. You look tired. You ___ rest. (should, should to, shoulds)
  2. After 'should', we use the base verb without 'to'.
  3. At this restaurant, you should ___ the arepas — they're amazing.
  4. It's very popular, so you should ___ a table in advance.
  5. The dessert is huge. You ___ order two! (negative)
  6. Find the mistake: You should to try the fish.
  7. You rest. — Give advice with 'should'.
  8. You should arrive early. — Turn it into a question.
  9. Give one suggestion for your favourite restaurant. (Use 'You should...')

Listening

Kenji: I want to take Sofía to a nice restaurant. Any advice?

Ben: Yes! You should book a table early.

Kenji: Good idea. What should we order?

Ben: You should try the local dish. It's the best!

Kenji: And dessert?

Ben: You shouldn't skip it! But you shouldn't forget to leave a tip.

Reading

Ben loves giving advice about restaurants. 'Here are my five rules,' he says. 'You should book a table. You should arrive early. You should try the local dish. You should ask the waiter for a recommendation. And you should always leave a tip!' Kenji writes it all down. 'What shouldn't I do?' he asks. Ben smiles. 'You shouldn't order too much — save room for dessert!'

  • Ben says you should try the local dish.
  • Ben says you shouldn't leave a tip.
  • You shouldn't order too much food.
