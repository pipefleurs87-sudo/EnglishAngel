Would like — Wishes & Polite Offers

A1 — grammar, listening, reading, writing

Common mistake: “I would like go to Japan.” — Correct: “I would like to go to Japan.”

We use 'would like' (I'd like) to express wishes and make polite offers; would like + to + verb, or + a noun.

would like (I'd like)
FormExample
+ to + verbI would like to travel
+ nounI'd like a coffee
QuestionWould you like...?
Short answerYes, I would. / No, I wouldn't.
  • I would like to go to Japan next vacation.
  • Sofía would like a coffee, please.
  • I wouldn't like to live in a very cold country.
  • Ben wouldn't like to eat spicy food.
  • Would you like to watch a movie? What would you like for dinner?
  • Where would you like to travel next?
  • Would you like some tea? Yes, I would.
  • Would you like to dance? No, I wouldn't.

Practice

  1. I would like ___ go to Japan. (to, for, -)
  2. We say 'would like + to + verb' or 'would like + noun': 'I'd like to travel' / 'I'd like a coffee'.
  3. What would you like ___ dinner tonight?
  4. Would you like ___ watch a movie this weekend?
  5. I would like to go to a place ___ I can relax and see the sea.
  6. Find the mistake: I would like go to the beach.
  7. You would like to dance. — Turn it into a question.
  8. I want to travel to Japan. — Rewrite it politely with 'would like'.
  9. Where would you like to travel next vacation?

Listening

Ben: Kenji, would you like to watch a movie this weekend?

Kenji: Yes, I would! What would you like to see?

Ben: I'd like to see the new adventure film.

Kenji: Perfect. And would you like to eat something after?

Ben: Yes. I would like to try that new restaurant.

Kenji: Great! I'd like some Japanese food.

Reading

It's Friday, and everyone has plans. 'I would like to travel to Japan,' says Kenji, 'because I'd like to see my family.' Sofía smiles. 'I would like to relax. I'd like a quiet weekend at home.' And Ben? 'I would like to try every restaurant in the city! Would you like to come with me?' 'No, thanks,' laughs Sofía. 'I wouldn't like to eat that much!'

  • Kenji would like to travel to Japan.
  • Sofía would like a busy, noisy weekend.
  • Ben would like to try every restaurant.
